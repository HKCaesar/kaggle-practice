## Homesite Competition
 [简书](http://www.jianshu.com/p/5709fa18cdb4 "悬停显示")  
 赛题：保险公司根据以往的用户信息和保险报价，以及最后是否购买了保险这些数据，希望最后得到一个分类模型，根据用户信息和报价，预测该用户是否会购买该单保险。  
 环境：numpy, scikit-learn, pandas, xgboost
